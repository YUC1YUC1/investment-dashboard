## 更新方式

1. 解壓縮。
2. 將 `app.py`、`requirements.txt`、`README.md` 與 `.streamlit` 覆蓋到原 GitHub repository。
3. 在 GitHub Desktop 按 **Commit to main**。
4. 按 **Push origin**。
5. Streamlit Cloud 會自動重新部署。
